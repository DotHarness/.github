## About DotCraft

DotHarness builds DotCraft, an open-source Agent Harness powered by .NET 10 and a Unified Session Core. It gives developers one observable workspace where agents can understand a project, run across multiple clients, and keep sessions, approvals, skills, and tools connected under the same workflow.

DotCraft is designed for teams and individuals who want AI collaboration to live inside their real development environment, not in a separate chat window.

## What You Can Build With It

- Project-level AI workspaces that persist context across sessions.
- Desktop, CLI, TUI, IDE, bot, and automation entry points connected to the same session core.
- Observable agent workflows with approvals, traces, tool calls, and workspace configuration.
- External channels and custom clients through AppServer, Python SDK, and TypeScript SDK integrations.
