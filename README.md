## 🎯 Who We Are

**DotCraftDev** is an open-source organization focused on **AI-powered development tools**, dedicated to making developers' workflows smarter, more efficient, and more secure.

We believe: **Great tools should be lightweight, open, and extensible**.

---

## 🌟 Core Project

### [DotCraft](https://github.com/DotCraftDev/DotCraft) — A Lightweight .NET Assistant

**A lightweight .NET assistant crafting your intelligent workspace**

**Key Features:**
- 🪶 **Lightweight & Minimal** - Single-file deployment, no complex dependencies
- 🔌 **Dual Protocol Powered** - ACP connects editors, MCP extends toolchains
- 🎮 **Full Scenario Coverage** - JetBrains IDEs, Unity, CLI, Chat Bots
- 🔒 **Secure & Controllable** - Multi-layer security with approval flow for high-risk operations
- 🐳 **Sandbox Isolation** - Support OpenSandbox containerized execution

**Tech Stack:** `.NET 10` `C#` `ACP` `MCP` `Unity Editor`
